LEARNING PATH

从“会用模型”走向“会做 AI 系统”

学习顺序不是概念堆砌，而是每一段都能做出一个更完整的东西。

STEP 1

模型基础认知

补齐 token、context、采样参数、幻觉和对齐等基本概念。

STEP 2

API 与结构化输出

理解消息结构、工具调用、流式返回和 JSON 输出稳定性。

STEP 3

RAG 与知识增强

做一个自己的资料问答系统，理解检索、切片和重排的真正价值。

STEP 4

Agent 与自动化

给模型接上工具和外部系统，让它真正开始处理任务而不是只回答问题。
